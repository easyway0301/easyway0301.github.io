# Seeed XIAO ESP32-S3 with MicroPython

本專案為 **Seeed Studio XIAO ESP32-S3** 的 MicroPython 入門教學文件，  
內容以「實際操作流程（SOP）」為導向，適合初學者、教學示範與自學使用。

---

## 📌 專案內容

本教學將帶你完成以下流程：

1. 認識 XIAO ESP32-S3 開發板（BOOT / RESET 按鈕）
2. 安裝 MicroPython 開發環境（Thonny）
3. 將 ESP32-S3 連接至電腦
4. 燒錄 MicroPython 韌體至開發板

## 🧰 使用設備與工具

- Seeed Studio **XIAO ESP32-S3**
- USB 傳輸線
- 電腦（Windows / macOS / Linux）
- **Thonny IDE**
  - 官方網站：https://thonny.org

---

## 🐍 為什麼選擇 MicroPython？

- 語法簡單、學習曲線低
- 適合教學與快速原型開發
- 可直接在板子上即時執行與測試程式
- 非常適合 ESP32 系列開發板
