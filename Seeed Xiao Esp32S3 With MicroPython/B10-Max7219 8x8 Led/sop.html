MAX7219 8x8 LED（SPI）

本 SOP 示範如何在 Seeed XIAO ESP32S3 上， 使用 MAX7219 8x8 LED 點陣模組， 透過 MicroPython + SPI 顯示簡單文字（HELLO）。

一、硬體接線說明

MAX7219XIAO ESP32S3
VCC5V 或 3V3
GNDGND
DINGPIO 9 (MOSI)
CSGPIO 8
CLKGPIO 7 (SCK)

二、程式功能說明

  • 使用 SPI 初始化 MAX7219
  • 設定亮度與清空畫面
  • 自建英文字母點陣資料
  • 逐字顯示 HELLO
  • 不使用 while 迴圈

三、MicroPython 程式碼（逐行註釋）


from machine import Pin, SPI
import max7219
import time

spi = SPI(1, baudrate=10_000_000, sck=Pin(7), mosi=Pin(9))
cs = Pin(8, Pin.OUT)

display = max7219.Matrix8x8(spi, cs, 1)
display.brightness(5)

for x in range(32, -40, -1):
    display.fill(0)
    display.text("HELLO", x, 0, 1)
    display.show()
    time.sleep(0.1)

四、顯示效果

8x8 點陣會依序顯示： 

H → E → L → L → O
